# Data Directory

This folder contains JSON files with German vocabulary cards for development and testing.

## Files

- **nouns.json** - German noun vocabulary
- **verbs.json** - German verb vocabulary
- **phrases.json** - German phrase vocabulary

These files will be used to:
1. Test the import functionality
2. Provide realistic mock data during UI development
3. Seed the app with initial vocabulary

## Note

These files are committed to the repository as sample/seed data for the application.
